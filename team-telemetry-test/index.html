Installation

  • First, download one of the Team Telemetry Versions
  • Unzip the File and double click the "TT......Setup.exe"
  • When installing / starting Team Telemetry for the first time, you will be asked to install install 2 runtime library / frameworks from Microsoft
  • After the installation of the runtime libraries, Team Telemetry will start
